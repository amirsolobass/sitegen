Maze Solver

Generates random mazes and solves them in real time with graphical visualization. Built with a custom drawing library on top of Python's tkinter.

Features

  • Procedural maze generation
  • Animated pathfinding solver with visual step-by-step rendering
  • Custom graphics library (cells, walls, lines) built on tkinter
  • Interactive window with menu system
  • Unit test suite for the maze and cell logic

How It Works

The maze is represented as a grid of cells, each with four walls that can be open or closed. Generation carves passages using a recursive algorithm, producing a connected maze with a unique solution path.

The solver then traverses the maze using depth-first search, drawing the explored path live in the window. Backtracking is shown in a different color, so you can see the algorithm think.

The graphics layer draws directly to a tkinter Canvas, with custom abstractions for lines and cells that keep the algorithm code clean.

Tech Stack

  • Language: Python
  • Graphics: tkinter (custom drawing library)
  • Algorithms: Recursive backtracking (generation), DFS (solving)
  • Testing: Python unittest
