Tubeify

A media file server and CDN application — upload videos and images, process them server-side with FFMPEG, and serve them efficiently via AWS S3 and CloudFront. Built as part of Boot.dev's file storage and CDN course.

Features

  • Video and image upload with metadata extraction
  • Server-side transcoding and thumbnail generation via FFMPEG
  • Cloud storage with AWS S3 and CloudFront CDN delivery
  • JWT-based authentication
  • SQLite database for media metadata
  • Sample media download script for local development

How It Works

Uploads hit the Bun HTTP server, which validates the JWT and passes the file to an FFMPEG pipeline for transcoding (video → optimized format, images → resized thumbnails). Processed files are pushed to S3, and CloudFront serves them to clients via a CDN edge — reducing latency and keeping origin load low.

Bun's native file APIs make streaming uploads and responses fast without needing a separate streaming library.

Tech Stack

  • Runtime: Bun
  • Language: TypeScript
  • Database: SQLite (via Bun's native SQLite)
  • Media processing: FFMPEG
  • Cloud storage: AWS S3
  • CDN: AWS CloudFront
  • Auth: JWT (jsonwebtoken)
