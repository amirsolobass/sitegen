gatoRSS

A command-line RSS feed aggregator with user management, feed subscriptions, and a scheduled background aggregation loop — all backed by PostgreSQL.

Features

  • Multi-user support with persistent sessions
  • Subscribe and unsubscribe from RSS feeds
  • Scheduled aggregation: fetch and store posts on a configurable interval
  • Browse aggregated posts from your followed feeds
  • Command registry pattern: easily extensible with new CLI commands

How It Works

gatoRSS is built around a command dispatcher — each CLI argument maps to a registered handler function. User state is persisted in PostgreSQL and accessed through Drizzle ORM, keeping the database layer typed and migration-friendly.

Feed content is fetched over HTTP and parsed from XML using fast-xml-parser. The aggregation loop runs on a configurable timer, polling each followed feed and writing new items to the database without duplicates.

Tech Stack

  • Language: TypeScript (Node.js)
  • Database: PostgreSQL via Drizzle ORM
  • RSS Parsing: fast-xml-parser
  • Auth: User session stored in DB
  • Runner: tsx (TypeScript execution)
