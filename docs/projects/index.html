Projects

Everything here was built by me. Vibe coding was kept to a minimum.

  • **Chess Engine** — Full chess game with a minimax AI opponent. Enforces every rule: castling, en passant, promotion, repetition. Zero dependencies.
  • **Server** — Production-ready REST API with JWT auth, argon2 password hashing, Drizzle ORM, webhooks, and a Vitest test suite.
  • **PCA** — AI agent powered by Google Gemini. Reads files, writes code, and executes scripts through a function-calling loop.
  • **Claude Assist** — PCA rebuilt on the Anthropic Claude API, with tool use, multi-turn agentic loops, and prompt caching.
  • **gatoRSS** — CLI RSS feed aggregator with user management, feed subscriptions, and a scheduled aggregation loop backed by PostgreSQL.
  • **Tubeify** — Video and image upload server with FFMPEG transcoding, S3 + CloudFront delivery, and JWT auth.
  • **life-xp** — A gamified life-tracking web app. Log experiences, earn XP.
  • **Maze Solver** — Generates and solves mazes with real-time graphical visualization using a custom tkinter drawing library.
  • **Pokedex** — Interactive REPL for exploring Pokémon data with a caching layer, persistent state, and modular command handlers.
  • **CTN** — Music theory CLI that converts chord names — including slash chords and extended harmonies — to their constituent notes.
  • **BookBot** — Text analysis CLI: word count and character frequency table for any document.
  • **Astroids** — A fully playable Asteroids clone. Asteroid splitting, shield mechanic, power-ups, high score system.
