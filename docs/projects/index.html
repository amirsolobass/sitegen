Projects

Everything here was built by me. Vibe coding was kept to a minimum.

  • **Chess Engine** — Full chess game with a minimax AI opponent. Enforces every rule: castling, en passant, promotion, repetition. Zero dependencies. *JavaScript*
  • **Server** — Production-ready REST API with JWT auth, argon2 password hashing, Drizzle ORM, webhooks, and Vitest test suite. *TypeScript, Express 5, PostgreSQL*
  • **PCA — Personal Coding Assistant** — AI agent powered by Google Gemini. Reads files, writes code, and executes scripts through a function-calling loop. *Python, Gemini API*
  • **Claude Assist** — PCA rebuilt on the Anthropic Claude API, with tool use, multi-turn agentic loops, and prompt caching. *Python, Claude API*
  • **gatoRSS** — CLI RSS feed aggregator with user management, feed subscriptions, and a scheduled aggregation loop backed by PostgreSQL. *TypeScript, Drizzle ORM*
  • **Tubeify** — File server and CDN learning project: video/image uploads, FFMPEG transcoding, S3 + CloudFront delivery, JWT auth. *TypeScript, Bun, SQLite, AWS S3*
  • **life-xp** — A gamified life-tracking web app. Log experiences and earn XP. *React 19, TypeScript, Vite*
  • **Maze Solver** — Generates and solves mazes with real-time graphical visualization using a custom tkinter drawing library. *Python*
  • **Pokedex** — Interactive REPL for exploring Pokémon data with a caching layer, persistent state, and modular command handlers. *TypeScript*
  • **CTN — ChordToNote** — Music theory CLI that converts chord names (including slash chords and extended harmonies) to their constituent notes. *Python*
  • **BookBot** — Text analysis CLI that counts words and character frequencies in any document. My first Boot.dev project. *Python*
  • **Astroids** — A browser-based Asteroids clone. *Python*
