life-xp

A gamified life-tracking web application. Log real-world experiences and earn XP — turning everyday activities into a progression system.

Features

  • Log experiences and activities with XP rewards
  • Track progress over time
  • React 19 with concurrent rendering
  • Fast development and build via Vite
  • Fully typed with TypeScript
