Chess Engine

A fully functional chess game with an AI opponent built in vanilla JavaScript — zero dependencies. The AI uses minimax search with alpha-beta pruning, quiescence search, and piece-square table evaluation. Enforces every rule: castling, en passant, pawn promotion, threefold repetition, and the 50-move rule.

Tech: JavaScript, Minimax, Alpha-Beta Pruning, Docker

Server

A production-ready REST API built with Express 5, featuring JWT authentication, argon2 password hashing, Drizzle ORM with PostgreSQL, webhook handling, admin metrics, and a full test suite with Vitest.

Tech: TypeScript, Express 5, PostgreSQL, Drizzle ORM, JWT, Argon2

PCA — Personal Coding Assistant

An AI agent powered by Google's Gemini API. Accepts a natural-language prompt and autonomously reads files, writes code, and executes Python scripts through a function-calling architecture with up to 20 iterations.

Tech: Python, Google Gemini API, Function Calling
